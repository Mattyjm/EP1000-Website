3D modelling/CAD


In this lesson, we learned about basic operations in Fusion360/Inventor. I used Inventor (for the majority) as I was more comfortble with it.

Exercise 1 : Name Tag

The first exercise we did was making a nametag. One new feature i learned was the "Emboss" feature that allowed me to choose a profile (in this case a text box), and emboss it on any surface i wanted.

Steps Photos
Sketch out basic profile.

Dimensions are 70 by 30mm, with a 4mm keyhole. Internal line is made using Offset Tool, with 1.5mm offset.
Extrude to create the solid.

The external lip is a 4mm extrusion, internal is 1.5mm.
Using a text box, I typed my name in, and use the Emboss Tool to extrude the profile by 2mm.

Exercise 2: Lego Brick

The second model we did was making a Lego brick via this video. This used a command that was new to me, which was "Shell". This removed material from the center, and created a wall of defined thickness around the whole solid.

Exercise 3: Cube Stack

Next, we were challenged to make a stack of cubes. There are many ways of doing it.
I did it systematically, layer by layer (as seen below), so as to reduce the number of sketches and extrusions needed.
