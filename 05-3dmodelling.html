3D Modelling

This page will document the 3D printing process, as well as how I was able to to design files for 3D printing & get the 3D printed object. This page will show how I created the mug and hex nut.

Exercise 1: Mug

The first exercise we did was making a mug. Since we were able to use our own design I used my own and made a rocket-looking mug.

Steps Photos
I started by sketching the profile by using the Arc tool to create a simple design, and a arc at the base (as per requirements).

Diameter of mug is 25mm, and height is 80mm.
I used the Revolve Tool to create the solid.
I then used the Shell tool to make the internal cavity.
Wall thickness is 2.5mm (as per requirements).
The mug handle profile is a simple sketch, by creating the external side, and using the Offset Tool to create the internal side.

Offset is 3mm from the external side.
Using Extrude Tool (symmetrical extrude), the mug handle is created.

Thickness of the handle is 6mm (as per requirements).
Use of Fillet Tool to remove sharp edges.
Use of Chamfer Tool to remove sharp edges.
