Here, I will document the assignments where the 3D printer was used.
We used the 3D printer to make a model chess knight, as well as a phone stand.

Assignment: Knight

The assignment we were given to do was making a chess knight. On the right you can see the final model in Fusion360.

Steps Photos
First, I imported the image of the knight into Fusion360 as a canvas.

This will allow me to trace the image, and create the outline easily. I then traced out the base.
Next, I used the Revolve tool to create the circular base.
I then used the Shell tool to make the base hollow (as per instructions).
Using the canvas, I traced out the top of the knight.
After using the Extrude Tool (symmetrical extrude), the top of the knight is created.
Added eyes using the Hole Tool.
Added eyes on the other side using the Mirror Tool.
Creation of a tangent place to make the ears.
Use of Fillet Tool to remove sharp edges.
Use of Chamfer Tool to remove sharp edges.

3D Printing (Chess Knight)

I then ran my model through the slicing software (Cura) to be sent to the 3D printer (Ultimaker 2+).


Settings used:
Layer Height: 0.3mm, Infill Density: 10%, Print Speed: 60mm/s

Assignment: Phone Stand

The next assignment we were given was to create a phone stand. I created a simple design; so as to reduce the need for supports and wasted material.

Steps Photos
I started out with a simple sketch of the phone stand. I wanted it be tilted at about 65-70 degrees.

Thickness of the stand is 2.5mm.
I then used the Extrude Tool (symmetrical extrude) to create the shape.

Extrusion is 50mm total.
Use of Fillet Tool to remove sharp edges.

3D Printing (Phone Stand)

I then ran my model through the slicing software (Cura) to be sent to the 3D printer (Ultimaker 2+). (insert phonestand when get it back)


Settings used:
Layer Height: 0.3mm, Infill Density: 10%, Print Speed: 60mm/s
